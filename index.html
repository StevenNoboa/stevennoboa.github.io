Proyecto final Data Science - Bootcamp

Proyecto real en colaboración con la empresa Several Energy, el cual concluyó con éxito para nuestro equipo, resultando ganadores del mismo. EL Proyecto se centra en la creación, desarrollo y aplicación de un proceso de automatización de extracción de datos y autorrellenado de formularios para el equipo de ventas y su aplicación web. Reporte analítico con PowerBI y creación de una base de datos relacional.

Análisis del precio de coches de segunda mano en España

Conoceremos cómo se establece la variable precio respecto a otras: tipo de combustible, año de producción, kms, potencia, provincia... Entnderemos los factores, relaciones y distribuciones del mercado.

Proyecto Machine Learning Bank Customer Churn Detection

Analizar y predecir el comportamiento del cliente, especialmente si es probable que abandonen el banco. Identificar factores que contribuyen a la rotación y ayudar en el desarrollo de estrategias para retener a los clientes.

Proyecto de base de datos con AWS RDS, Flask y OpenAI

Creación de una aplicación cuyo objetivo principal es conectar una base de datos SQL alojada en AWS con Langchaim de OpenAI utilizando Flask (python). La aplicación permite a los usuarios subir archivos en formato de texto plano (.txt), scripts de Python (.py) y cuadernos Jupyter (.ipynb). Estos archivos se procesan utilizando la tecnología Langchaim de OpenAI para realizar tareas como resúmenes automáticos o incluso generar chistes basados en el contenido de los archivos.

Api-BBDD-Modelo ML

Desarrollar una API con Flask para desplegar un modelo de machine learning que predice ventas a partir de gastos en marketing en televisión, radio y periódicos. La API permitirá que una web interna, desarrollada en AngularJS, consuma el modelo y se comunique con una base de datos para ingestar nuevos registros y reentrenar el modelo.
